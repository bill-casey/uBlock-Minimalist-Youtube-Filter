# uBlock-Minimalist-Youtube-Filter
Filters for uBlock Origin which remove the majority of the recommendation spam YouTube constantly puts in front of you. 
This blocklist grew organically over time as I tried to cut down on all the things YouTube kept shoving in my face. Feedback and enhancements are 100% WELCOME. 

You can either import this list using the Import... option in uBlock, or copy the lines in filters.txt and paste them into your My Filters section.
